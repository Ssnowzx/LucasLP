Organize e acompanhe todas as tarefas do negocio. Arraste tarefas entre colunas para atualizar seu status.

💡 Como usar
  • Arraste tarefas entre colunas (desktop e mobile)

Metas & Objetivos

Defina metas claras para seu negocio e acompanhe o progresso em tempo real.

💡 Dicas
  • SMART: Metas devem ser Especificas, Mensuraveis, Alcancaveis, Relevantes, Temporizadas
  • Botoes de incremento: Use +500, +1k, +5k para atualizar progresso rapido
  • Categorias: Organize por tipo (receita, clientes, ads, etc)
  • Visual: A barra fica verde quando completa a meta!

Notas Compartilhadas

Comunique-se rapidamente com seu socio. Todas as notas ficam salvas no seu dashboard.

💡 Dicas
  • Enter: Pressione Enter para enviar sua nota (Shift+Enter para quebra de linha)
  • Privado: Apenas voces dois veem estas notas (nao sao compartilhadas online)
  • Avatar: L = Lucas, R = Rodrigo — identifica quem escreveu
  • Timestamp: Cada nota registra quando foi criada

Facebook Ads

Gerencie e acompanhe suas campanhas de trafego pago. Conecte sua API ou registre dados manualmente.

💡 Dicas
  • API Key: Insira seu token do Facebook Marketing API para sincronizar dados automaticamente
  • Manual: Voce pode adicionar campanhas manualmente enquanto nao conecta a API
  • ROAS: Retorno sobre investimento em ads — acima de 3x e saudavel
  • Seguranca: A chave da API fica salva localmente no seu navegador. Use Supabase para mais seguranca
